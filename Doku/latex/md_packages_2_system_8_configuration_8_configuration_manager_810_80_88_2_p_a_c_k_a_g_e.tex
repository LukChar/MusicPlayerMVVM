Provides types that support using XML configuration files ({\ttfamily app.\+config}). This package exists only to support migrating existing .NET Framework code that already uses System.\+Configuration. When writing new code, use another configuration system instead, such as \href{https://www.nuget.org/packages/Microsoft.Extensions.Configuration/}{\texttt{Microsoft.\+Extensions.\+Configuration}}.\hypertarget{md_packages_2_system_8_configuration_8_configuration_manager_810_80_88_2_p_a_c_k_a_g_e_autotoc_md40}{}\doxysection{\texorpdfstring{How to Use}{How to Use}}\label{md_packages_2_system_8_configuration_8_configuration_manager_810_80_88_2_p_a_c_k_a_g_e_autotoc_md40}
